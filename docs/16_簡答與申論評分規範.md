# 簡答與申論評分規範

簡答與申論不可逐字比對。評分以核心概念命中為主，可接受同義詞，支援部分給分與矛盾檢查。

## Rubric Schema

每份 Rubric 包含 requiredConcepts、acceptedSynonyms、optionalConcepts、contradictions、maxScore、passingScore 與低信心人工確認門檻。正式答案仍保留於正式題庫，Rubric 不得覆蓋它。

## 評分流程

1. 讀取指定題目的正式 Rubric。
2. 比對核心概念與同義詞。
3. 回傳分數、命中／缺少概念、可選概念、矛盾、信心與回饋。
4. 答案過短、低信心、出現矛盾或沒有 Rubric 時，一律標示「需人工確認」。

Sprint 11 僅提供規則型 `evaluateShortAnswer` 工具，未建立簡答 UI、AI API 或自動覆蓋正式答案的流程。
