# 錯題複習與熟練度規範

- 狀態：newWrong、reviewing、highRisk、improving、mastered。
- 答錯會增加 wrongCount、歸零 consecutiveCorrect 並標為 highRisk。
- 錯題複習答對會增加 correctReviewCount 與 consecutiveCorrect；連續答對 2 次才標為 mastered。
- 答對一次不刪除錯題，mastered 題目保留紀錄但預設不進入 reviewWrong。
- reviewWrong 使用正式題庫題目並寫入 StudySession mode = reviewWrong；未來可在此紀錄上接間隔重複。

## Sprint 13.1－當日可複習規則

- `getReviewableWrongAnswers()` 與 `getWrongAnswerSummary().reviewableCount` 必須使用同一個 `isReviewableToday` 判斷，確保首頁入口與實際題組一致。
- highRisk、newWrong、reviewing 為可複習狀態；mastered 不可複習。
- improving 題目若 `lastReviewedAt` 是今天，不排入當日 reviewWrong；隔日可再次依規則排入。
- reviewWrong 以實際 reviewQuestions 計時，範圍為 5～20 分鐘，不使用 formal-exam 的 90 分鐘。
