# AI 題庫維護規範（Knowledge Governance）

## 目的

本文件定義 IFA Master Coach 所有題目、答案、解析與 AI
出題的來源及維護原則。

本規範優先於一般 AI 推論。

AI（ChatGPT、Codex、Gemini 或其他工具）不得自行猜測或修改正式答案。

------------------------------------------------------------------------

# 一、資料來源優先順序

## Priority 1（最高優先）

使用者提供之正式資料：

-   IFA 歷屆考古題
-   學校考卷
-   教師講義
-   術科考核資料
-   使用者整理之正式題庫

上述資料視為正式依據。

------------------------------------------------------------------------

## Priority 2

IFA 官方資料：

-   IFA Syllabus
-   IFA Curriculum
-   IFA 官方課程內容
-   IFA 官方公開資訊

------------------------------------------------------------------------

## Priority 3

IFA 常用權威教材，例如：

-   Salvatore Battaglia
-   Robert Tisserand
-   Joy Bowles
-   Jane Buckle

------------------------------------------------------------------------

## Priority 4

近年經同儕審查（Peer-reviewed）研究、官方研究或具公信力之芳療研究。

------------------------------------------------------------------------

# 二、答案規範

AI 不得：

-   自行猜測正確答案
-   自行覆蓋既有答案
-   因一般網路文章修改正式答案
-   因模型推論修改考古題答案

若來源衝突：

1.  保留原始題目。
2.  保留原答案。
3.  標記「需人工確認」。
4.  不得自行修正。

------------------------------------------------------------------------

# 三、解析規範

解析必須：

-   說明答案原因。
-   引用對應知識點。
-   可補充臨床背景。
-   不得與正式答案矛盾。

若屬 AI 補充內容，需與正式答案一致。

------------------------------------------------------------------------

# 四、AI 產生新題目

只有在沒有正式考古題時，才可依據：

-   IFA Syllabus
-   官方教材
-   權威教材

建立模擬題。

所有 AI 生成題目必須標示：

「模擬題（非歷屆試題）」

不得偽裝成正式考古題。

------------------------------------------------------------------------

# 五、題庫維護流程

新增題目：

原始 PDF → 人工確認 → JSON 題庫 → 知識點對應 → AI 解析 → 測試 → 正式使用

不得直接由 PDF 自動大量轉換為正式題庫。

------------------------------------------------------------------------

# 六、原始資料管理

原始資料存放：

private/考古題原始資料/

不得：

-   部署到 GitHub Pages
-   提交到公開 GitHub
-   修改原始 PDF

正式資料：

src/data/questions/ src/data/knowledge/

------------------------------------------------------------------------

# 七、Codex 工作規範

Codex 不得：

-   修改正式答案
-   重寫題庫
-   刪除題目
-   自動更新大量 JSON

若需修改題庫，必須依照本規範並取得使用者明確指示。

------------------------------------------------------------------------

# 八、ChatGPT 工作規範

ChatGPT 可：

-   比對資料來源
-   找出衝突
-   建議修正
-   協助建立解析
-   協助建立模擬題

不得將推論視為正式答案。

------------------------------------------------------------------------

# 九、最重要原則

本專案以「協助通過 IFA 認證」為目標。

當 AI 推論與正式資料衝突時：

永遠以使用者提供資料與 IFA 官方資料為優先。

不知道時必須明確標示「需人工確認」，不得猜測。
