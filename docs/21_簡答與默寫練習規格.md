# 簡答與默寫練習規格

## 目的

提供名詞解釋、簡答與默寫的最小自我檢核流程。使用者先輸入文字答案，再依參考答案與檢核點自行決定是否需要複習。

## 自我檢核與 Rubric

- 顯示參考答案、必要核心概念、可接受同義說法、常見矛盾點與通過分數。
- 缺少 Rubric 時顯示請依參考答案自我檢核的安全提示。
- 不做 AI 自動判分、語意相似度或完整申論批改。

## 紀錄與錯題

- 使用者選擇結果寫入 `writingPractice` StudySession，保存題目 ID 與正確／需要複習分類。
- 選擇需要複習時，沿用 wrongAnswerStore，只保存題目索引與統計，不保存完整題目內容。
- Recent Activity 以「簡答／默寫練習」顯示此模式。

## 題目來源與後續

目前僅提供 sample／needsReview 示範題，與正式題庫分離；未來 AI Coach 可讀取 self-check 結果，但不得改寫正式答案或取代使用者自我檢核。
