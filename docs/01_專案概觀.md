# 專案概觀

## 目標

建立 IFA Master Coach，提供 IFA
國際芳療師考試之模擬測驗、解析與學習平台。

IFA Master Coach 的核心目標不是讓使用者閱讀最多教材，而是透過科學化的練習、複習與回憶機制，在有限時間內提升通過 IFA 考試的機率。每日任務維持手感與降低中斷；每週任務累積題量與加強記憶；複習優先處理錯題、弱點與高風險知識點。題庫功能必須服務於提高通過率。

## MVP

-   Week1\~Week8
-   模擬考
-   成績
-   錯題解析
-   localStorage 保存
-   GitHub Pages 發布

## 技術

React、Vite、TypeScript、Tailwind、PWA、GitHub Pages。
